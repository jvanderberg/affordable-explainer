# Oak Park Housing Affordability Explainer

A scroll-driven visual explainer showing how housing supply, aging, household moves, targeted subsidy, and market-rate construction affect affordability in Oak Park, Illinois.

The analysis draws on the U.S. Census Bureau's American Community Survey, Illinois Housing Development Authority affordability lists, the Village of Oak Park's Strategic Vision for Housing, and published housing research.

## Development

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Run `npm test` for the production build and rendered-content check.

## Publishing

Pushes to `main` automatically build and deploy the static site through GitHub Pages.
